# ✨ Web AR Video Overlay Project (MindAR + A-Frame)

नमस्ते! आपके लिए एक बेहतरीन और एडवांस्ड **Web AR (Augmented Reality)** प्रोजेक्ट तैयार कर दिया गया है। 
इस प्रोजेक्ट की मदद से जब कोई आपकी शेयर की गई इमेज (Drawing/Sketch) को स्कैन करेगा, तो ठीक उसी इमेज के ऊपर (Overlapping) आपका सेट किया हुआ वीडियो प्ले होने लगेगा।

---

## 🚀 3 Simple Steps to Set Up Your AR / सेट-अप करने के 3 आसान तरीके:

### 1️⃣ अपना टारगेट माइंड फाइल (`targets.mind`) तैयार करें
Web AR को यह पता लगाने के लिए कि किस इमेज पर वीडियो प्ले करना है, एक `.mind` फाइल की जरूरत होती है। इसे आप 1 मिनट में फ्री में बना सकते हैं:
1. इस लिंक पर जाएं (MindAR Official Compiler): **[https://hiukim.github.io/mind-ar-js-doc/tools/compile](https://hiukim.github.io/mind-ar-js-doc/tools/compile)**
2. अपनी जो इमेज (राशू/Radhu का स्केच) आपने शेयर की है, उसे वहाँ **"Drop images here"** बॉक्स में अपलोड करें।
3. **Start** पर क्लिक करें। प्रोसेस पूरा होने के बाद **Download** बटन पर क्लिक करें।
4. डाउनलोड हुई फाइल का नाम `targets.mind` रखें और उसे इस फोल्डर (`web ar`) में पहले से मौजूद `targets.mind` की जगह रिप्लेस (overwrite) कर दें।

### 2️⃣ अपना वीडियो (`video.mp4`) फोल्डर में डालें
1. आप जो भी वीडियो प्ले करवाना चाहते हैं, उसका नाम `video.mp4` रखें।
2. उस वीडियो को इस फोल्डर (`web ar`) में डाल दें (पहले से मौजूद `video.mp4` फाइल को रिप्लेस कर दें)।
*💡 **Tip:** बेहतरीन परफॉरमेंस के लिए वीडियो का फाइल साइज कम (10MB-15MB से कम) रखें और उसका Aspect Ratio इमेज के जैसा (Vertical 3:4 या 9:16) रखें।*

### 3️⃣ प्रोजेक्ट को चलाएं (Run & Test)
मोबाइल कैमरे और AR फीचर्स को चलाने के लिए ब्राउज़र को सुरक्षित कनेक्शन (HTTPS या Local Server) की आवश्यकता होती है। आप इसे दो तरीकों से चला सकते हैं:

**A. अपने कंप्यूटर पर टेस्ट करने के लिए (Local Test):**
- अगर आप VS Code का इस्तेमाल कर रहे हैं, तो **Live Server** एक्सटेंशन इंस्टॉल करें और `index.html` पर राइट-क्लिक करके "Open with Live Server" चुनें।
- अपने मोबाइल फोन से अपने कंप्यूटर का लोकल IP एड्रेस खोलकर टेस्ट करें।

**B. इंटरनेट पर फ्री में पब्लिश करने के लिए (Free Online Publishing - 100% Recommended):**
ताकि दुनिया में कोई भी अपने मोबाइल से इसे स्कैन कर सके, आप इस पूरे फोल्डर को फ्री में होस्ट कर सकते हैं:
- **[Vercel](https://vercel.com)**, **[Netlify](https://netlify.com)** या **[GitHub Pages](https://pages.github.com)** पर जाएं।
- अपना यह `web ar` फोल्डर वहाँ ड्रैग और ड्रॉप करें।
- आपको एक फ्री लाइव लिंक (जैसे `https://your-ar-app.netlify.app`) मिल जाएगा। इसे मोबाइल में खोलें, कैमरा परमिशन दें और अपनी इमेज को स्कैन करें! 🎉

## 📂 Project Structure / फाइल्स की जानकारी

- `index.html` — मुख्य AR एप्लीकेशन कोड (इसमें MindAR, A-Frame और मोबाइल वीडियो ऑटो-प्ले अनलॉक करने के एडवांस्ड स्क्रिप्ट्स हैं)।
- `style.css` — प्रीमियम यूआई (UI), ग्लॉसमॉर्फिज़्म कार्ड्स और स्कैनिंग एनिमेशन के लिए डिज़ाइन।
- `targets.mind` — आपकी इमेज की ट्रैकिंग फाइल (यहाँ अपनी डाउनलोड की गई `.mind` फाइल डालें)।
- `video.mp4` — आपका वीडियो फाइल (यहाँ अपना वीडियो डालें)।

## 🛠️ Customization Tips / अपनी पसंद के बदलाव

अगर आपके वीडियो की लंबाई-चौड़ाई (Aspect Ratio) सेट करनी हो, तो `index.html` में इस लाइन को खोजें:
```html
<a-video src="#ar-video" position="0 0 0" width="1" height="1.35" rotation="0 0 0"></a-video>
```
- अगर वीडियो चौड़ा (Horizontal) है, तो `width="1.33" height="1"` कर सकते हैं।
- अगर वीडियो लंबा (Vertical) है, तो `width="1" height="1.35"` (या `1.5`) कर सकते हैं ताकि वह इमेज पर पूरी तरह फिट बैठे।
